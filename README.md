# 🌏 世界頭條 — 即時新聞

> 即時聚合 BBC、CNN、NPR、Al Jazeera 等 38 個全球新聞源，支援多語言翻譯。

## 功能

- 🌍 13 個地區分類 + 5 個主題分類
- 🌐 7 種語言即時翻譯
- 🔍 Google News 即時搜尋
- 🌙 深色模式
- 📱 PWA 支援（可離線使用）
- 📶 38 個新聞來源

## 開發

```bash
npm install
npm run dev
```

## 建置

```bash
npm run build
```

## 部署

推送後 GitHub Actions 自動構建並部署至 GitHub Pages。
